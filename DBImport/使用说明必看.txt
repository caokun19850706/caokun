可用于MYSQL、SQLITE、Oracle数据库之间的数据相互导入导出转换与数据迁移！

MYSQL to SQLITE
SQLITE to MYSQL
Oracle to SQLITE
Oracle to MYSQL
MYSQL to Oracle
SQLITE to Oracle

常用连接字符串：

MYSQL：
host=数据库地址;port=3306;database=数据库名称;uid=用户;pwd=密码

SQLITE：
Data Source=数据库路径;failifmissing=false

使用说明：

1、运行“DBImport.exe”，依次输入对应数据库链接字符串，点击“链接”，显示出数据表列表和提示“链接成功‘开始导数据’”；
（数据库链接From：为从导出源数据库连接字符串，数据库链接To：为导入目标数据库连接字符串）

2、从数据表中选择将要出的数据表信息，点击“开始导数据”。
